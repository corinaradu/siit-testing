-----------------------Class examples-----------------------
Import settings:
1. Go to Chrome -> Settings -> More -> Extensions -> Form Filler and click Options there
2. In Form Filler options go to Backup and Restore - click Import Settings and import the Hp_registration_fields_export.txt file in this folder
3. If you go again to Form Filler options - Custom Fileds - you should see the custom fields imported together with the regular expressions used there.
